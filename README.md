## Installation

```
git clone https://github.com/zirho/layout

cd layout

yarn

yarn start
```

## Responsive layout with

 * browser-sync server
 * auto refresh when files are updated
 * fixed header
 * sticky footer
 * responsive left sidebar

#### Customize
 * Header height: 50px;
 * Footer height: 50px;
 * Sidebar width: 200px;
